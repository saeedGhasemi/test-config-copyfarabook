# پلن: متادیتای کامل کتاب + آپلود رزومیبل

## ۱) گسترش اسکیمای دیتابیس (migration)

به جدول `books` این فیلدها اضافه می‌شوند:

| فیلد | نوع | توضیح |
|---|---|---|
| `book_type` | text | `authored` / `translation` / `compilation` / `edited` / `adaptation` |
| `original_title` | text | عنوان اصلی (برای ترجمه‌ها) |
| `original_language` | text | زبان متن اصلی |
| `publication_year` | int | سال انتشار |
| `edition` | text | نوبت چاپ/نسخه |
| `isbn` | text | شابک (اعتبارسنجی ۱۰/۱۳ رقمی) |
| `page_count` | int | تعداد صفحات نسخه چاپی |
| `categories` | text[] | دسته‌بندی‌های چندگانه |
| `subjects` | text[] | موضوعات/کلیدواژه‌ها |
| `subtitle` | text | عنوان فرعی |
| `series_name` | text | نام مجموعه |
| `series_index` | int | شماره در مجموعه |
| `contributors` | jsonb | لیست مشارکت‌کنندگان |

شکل `contributors`:
```json
[
  { "name": "نام", "role": "author|translator|editor|illustrator|foreword|coauthor|compiler", "user_id": "uuid?", "order": 1 }
]
```

ستون قدیمی `author` (text) برای سازگاری حفظ می‌شود و از روی اولین مولف ساخته می‌شود.

## ۲) کامپوننت مشترک متادیتا

ساخت `src/components/book-metadata/BookMetadataForm.tsx` که در هر سه نقطه استفاده می‌شود:
- صفحه آپلود ورد (`Upload.tsx`)
- ادیتور کتاب (`BookEditor.tsx` — تب «مشخصات»)
- صفحه انتشار (`Publish.tsx`)

شامل:
- فیلدهای ساده (عنوان، عنوان فرعی، توضیحات، نوع کتاب، سال، ناشر، شابک، تعداد صفحات، نوبت چاپ، مجموعه، عنوان اصلی، زبان اصلی)
- مدیریت چند مشارکت‌کننده (افزودن/حذف/ترتیب، با انتخاب نقش)
- چند دسته‌بندی و موضوع (chip input)
- اعتبارسنجی شابک

## ۳) صفحه آپلود ورد

- جایگزینی فرم ساده با `BookMetadataForm` (همان فیلدها قبل از تبدیل پر می‌شوند)
- فقط فیلدهای حیاتی اجباری (عنوان + حداقل یک مولف). بقیه قابل ادیت بعدی.

## ۴) ادیتور کتاب

افزودن تب «مشخصات کتاب» با `BookMetadataForm`، اتوسیو روی `books`.

## ۵) صفحه انتشار

نمایش/ویرایش کامل متادیتا قبل از انتشار، با چک‌لیست فیلدهای پیشنهادی پرنشده.

## ۶) دیالوگ پیش‌نمایش فروشگاه

افزودن بلوک «شناسنامه کتاب» در `BookPreviewDialog`:
- لیست مشارکت‌کنندگان به‌تفکیک نقش
- ناشر، سال انتشار، شابک، تعداد صفحات، نوبت چاپ، نوع کتاب، مجموعه، زبان/عنوان اصلی، دسته‌بندی‌ها

## ۷) آپلود رزومیبل ورد (resumable)

جایگزین `XMLHttpRequest` تک‌مرحله‌ای با پروتکل **TUS** که Supabase Storage پشتیبانی می‌کند:

- استفاده از کتابخانه `tus-js-client`
- اندپوینت: `${SUPABASE_URL}/storage/v1/upload/resumable`
- chunkSize ~6MB، retryDelays برای ازسرگیری خودکار پس از قطعی شبکه
- ذخیره `uploadUrl` در `localStorage` (به ازای فایل) تا بازگشت کاربر هم بتواند ادامه دهد
- بازنشانی نوار پیشرفت بر اساس `bytesUploaded/bytesTotal`
- پس از موفقیت، همان مسیر جاری ساخت `word_imports` و فراخوانی edge function تبدیل اجرا می‌شود

## ۸) Edge function تبدیل

`supabase/functions/word-import/index.ts` بعد از تبدیل، فیلدهای متادیتای ذخیره‌شده در `word_imports` (که از فرم گرفته شده‌اند) را روی رکورد `books` تازه‌ساخته اعمال می‌کند. برای این کار به جدول `word_imports` چند ستون متادیتا اضافه می‌کنیم (jsonb `metadata`) که فرم در آن سریالایز کند، و فانکشن آن را روی کتاب کپی کند.

## خروج‌ها

- صفحه `Reader` بدون تغییر (فقط مصرف‌کننده اطلاعات اضافه است؛ اگر لازم شد در گام بعد).
- بدون تغییر در منطق پرداخت/سهم درآمد.
